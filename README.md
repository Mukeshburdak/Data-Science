## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Mukeshburdak/Data-Science.git
cd Data-Science
```

### 2. (Optional) Create a Virtual Environment

```bash
# On Windows
python -m venv venv
venv\Scripts\activate

# On macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

> If `requirements.txt` is not yet available, install the core libraries manually:
> ```bash
> pip install pandas numpy matplotlib seaborn scikit-learn jupyter
> ```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Then open any `.ipynb` file from the `Machine learning/` folder to get started!
